## :arrow_forward: Rodar o projeto

- Clone o repositório em sua máquina:
    ```sh
    $ git clone git@github.com:vinicius-rech/react-typescript.git
    ```
- Instale as dependências do package.json
    ```sh
    $ yarn install
    ```
   ou:
    ```sh
    $ yarn
    ```
- Para watch dev:
    ```sh
    $ # yarn dev
    ```
- Para watch dev em LAN:
    ```sh
    $ # yarn dev --host
    ```
